
## Tech Stack

**Technology:** Laravel, Vue 3, Axios, Element Plus, Vite JS

## Installation

```bash
  composer require
  npm install
  php artisan migrate
  php artisan db:seed

```
## Run Locally

Run

```bash
  php artisan serve
  npm run dev
  npm run build - if you want to build the script to public folder
```
